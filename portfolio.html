Brewology

An interactive website that allows the user can take a personality quiz to find out what beer matches his/her personality. Additionally the website, allows the user to search for breweries (using the Yelp API) based on the location that they input.

Trivia Game

A fun Disney Lyric Trivia Game- the player is asked to answer 5 questions referring to lyrics from songs in Disney movies. The player is given 2 minutes to try to answer all the questions. Once the player submits his/her answer or the timer runs out, the results of the quiz are revealed.

Crystal Collector

This game generates a random number from 19 to 120 that the player has to try to get in order to win the game. Each one of the four crystals is given a random number between 1 and 12. The player must click on crystals to try to make their sums add up to equal the exact total of the random number. Additionally, the game tracks the player's wins and losses.

Physic Game

The player has to guess the random letter generated. They have a total of nine tries. The page keeps track of the wins, losses, and letters already guessed.

GifTastic

A website with the theme of TV shows. There is a list of a few TV shows that once clicked show the gifs from that TV show from the Giphy API. Additionally, the user can add a button of any TV show that they would like and if that button is clicked on, gifs will appear from that TV show.
